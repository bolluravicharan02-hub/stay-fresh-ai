# Software Requirements Specification (SRS)

## Vision
Build an AI-powered invoice automation platform for supermarket owners using VasyERP.

## Stakeholders
- Supermarket Owner
- Store Staff
- Administrator
- AI Agent

## Functional Requirements
- Upload invoice (image/PDF)
- OCR extraction
- Structured invoice parsing
- Supplier detection
- Product extraction
- Barcode lookup
- Human review
- VasyERP submission
- Audit logs

## Non-Functional Requirements
- Production-ready
- Modular
- Secure
- Extensible
- Testable
